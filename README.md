Description:
This project allows users to listen to music instantly by selecting a song from songs.txt list. Once a song is chose,
the program automatically opens the corresponding Youtube video in the user's default web browser. It provide a 
simple and convenient way to access and enjoy music without manually searching for song online
Feature:
- Select and play song directly
- Automatically open Youtube in the default web
- lightway and easy to use
- Runs on any system with python installed 
How to run:
Python main.py
